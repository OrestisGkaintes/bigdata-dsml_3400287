# Δήλωση χρήσης εργαλείων LLM

Για την εκπόνηση της εργασίας χρησιμοποιήθηκε ένας aggregator (API που φιλοξενεί
διάφορα μοντέλα LLM). Απλούστερα μοντέλα χρησιμοποιήθηκαν για πλοήγηση στο terminal
του WSL, εκτέλεση των jobs στον server και ανάκτηση των log files, αποτελεσμάτων και
χρόνων εκτέλεσης. Πιο σύνθετα μοντέλα (Claude, ChatGPT) χρησιμοποιήθηκαν για την
υπόδειξη ενός διαγράμματος ροής εργασίας και την παραγωγή πρόχειρου κώδικα, καθώς και
για τη βελτίωση και την αναγνωσιμότητά του. Η επεξεργασία των κύριων σημείων της
λογικής κάθε query έγινε από εμένα.

| Ενότητα εργασίας | α) Εργαλείο | β) Έκταση χρήσης | γ) Τι έγινε από εμένα χωρίς LLM |
|---|---|---|---|
| Πλοήγηση στο terminal / εκτέλεση jobs | Aggregator (απλά μοντέλα) | Υπόδειξη εντολών `spark-submit`, `kubectl` και bash για εκτέλεση και ανάκτηση log files, αποτελεσμάτων και χρόνων | Η πραγματική εκτέλεση των εντολών, ο έλεγχος ορθότητας της εξόδου και η διόρθωση σφαλμάτων εκτέλεσης |
| Κώδικας Query 1–4 | Claude, ChatGPT | Παραγωγή πρόχειρου αρχικού σκελετού κώδικα και υπόδειξη του διαγράμματος ροής εργασίας | Επεξεργασία και επαλήθευση των κύριων σημείων της λογικής (parsing, aggregations, joins, formula per-capita), προσαρμογή στα πραγματικά schemas των datasets |
| Βελτίωση/αναγνωσιμότητα κώδικα | Claude, ChatGPT | Αναδιάρθρωση σε συναρτήσεις, σχόλια, καθαρισμός | Έλεγχος ότι οι αλλαγές διατηρούν την ορθότητα του αποτελέσματος |
| Κείμενο αναφοράς | Claude, ChatGPT | Βοήθεια στη σύνταξη και διατύπωση τμημάτων του σχολιασμού | Η ερμηνεία των μετρήσεων, τα συμπεράσματα και η τελική επιμέλεια |
